If creating visual artifacts (slides, mocks, throwaway prototypes, etc), copy assets out and create static HTML files for the user to view. Reference `colors_and_type.css` for tokens, `assets/` for logos and icon usage, and `ui_kits/wiseapp/` for component-level recreations.

If working on production code, you can copy assets and read the rules here to become an expert in designing with this brand. The brand language is operational, terse, factual; surfaces are white-on-gray with hairline borders and near-flat shadows; brand accent is amber `#f59e0b` used sparingly; icons come from `lucide-react`/`lucide-static`; copy is Russian-first.
